Pi Day 2022, Approximating Pi with π (3) Algorithms

For Pi Day 2022 (3/14/22), I created 3 different algorithms to estimate the value of Pi (~3.14159265358979323646433).

  • The first algorithm uses a large number of random coordinate points, and using the triangle formed by the x and y value of the point to see if the point would be in the first quadrant of the circle or not. The ratio of points in the circle to the total number of points, is approximately π/4.
  • The second algorithm uses a principle of Calculus called a Riemann Sum (using a large number of infinitely small rectangles to find the area under a curve) to aproximate the area of a semi-circle with radius 1, this finds an aproximation of π/2.
  • The third algorithm uses the physics of elastic colisions between 2 blocks of different masses to approximate pi, 3Blue1Brown has a great explainer on why this works

USB Reader Plugin for Hobbits

The USB Reader plugin for hobbits is a tool for reading data from USB devices and importing the data into Hobbits, a bitwise analysis and manipulation tool. I wrote this project as a part of my internship with Software, Magic, & Rainbows in Summer 2021. Using the Libusb library for C/C++ and QT C/C++ framework, I created this plugin that is currently available by default for the latest version of Hobbits for macOS and Linux.
